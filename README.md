# README

## Setup

0. Install ruby (>=3.0.0).

1. Install rails.

```bash
$ gem install rails
```

2. Get dependencies by running bundle install and npm install in the app directory.

```bash
$ bundle install
$ npm install
```

3. Run database migrations.

```bash
$ rails db:migrate
```

4. Run the server.

```bash
$ rails server
```
